# DeepLearning-NLP
This repository will apply deep learning methods to deal with natural language processing problems. I will use the biomedical dataset [MIMIC-III](https://mimic.mit.edu/mimicdata/whatsnew/) to train my model.

## Requirements:
* python (3.5.5)
* pandas (0.22.0)
* numpy (1.14.3)
* tensorflow (1.8.0)
