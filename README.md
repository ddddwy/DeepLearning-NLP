# DeepLearning-NLP
This repository will apply deep learning methods to deal with natural language processing problems. 

## Requirements:
* python (3.5.5)
* pandas (0.22.0)
* numpy (1.14.3)
* tensorflow (1.8.0)
* sklearn (0.0)
* scikit-multilearn (0.0.5)
